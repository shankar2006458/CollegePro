# 🎓 College Survival App

A modern all-in-one student productivity dashboard to manage attendance, timetable, assignments, exams, fees, and profile in one clean responsive web app.

## 🚀 Live Demo
🔗 https://shankar2006458.github.io/CollegePro/

---

## ✨ Features

### 📊 Smart Dashboard
- Live attendance percentage
- Upcoming exams counter
- Pending assignments tracker
- Fees due overview
- Today’s classes summary
- Alerts & reminders section

### 👤 Student Profile
- Save personal details
- Name, department, semester
- Roll number, college, email
- Greeting updates with first name

### 📚 Attendance Tracker
- Mark Present / Absent instantly
- Subject-wise attendance %
- Progress bars
- Need more classes calculator
- Attendance chart

### 📖 Subject Manager
- Add / Edit / Delete subjects
- Teacher names support
- Auto delete linked assignments/exams/timetable

### 🗓️ Timetable Planner
- Weekly class schedule
- Add class day/time/room
- Edit timetable entries
- Live time preview

### ✅ Assignments Manager
- Add assignment deadlines
- Priority levels
- Pending / Completed filters
- Completion stats chart

### 📝 Exams Section
- Add exam dates
- Countdown badges
- Notes support

### 💳 Fees Tracker
- Add fee records
- Due date reminders
- Mark as paid
- Due amount overview

### 🔔 Smart Alerts
- Low attendance warnings
- Upcoming deadlines
- Exam reminders
- Fee due notifications
